## Core Resources

The API is organized around these main resources:

- **Solicitud** - Transport request management with approval workflows
- **Mision** - Mission lifecycle from creation to completion
- **Vehiculo** - Vehicle fleet management and plate lookups
- **Motorista** - Driver profiles and availability
- **Catalogs** - Configurable statuses, types, and classifications
