---
title: 'Get Driver'
api: 'GET /motorista/{id}'
description: 'Retrieve detailed information about a specific driver'
---

Retrieves complete information for a specific driver, including profile data and driver-specific details.

## Endpoint

```
GET /motorista/{id_perfil}
```

## Path Parameters

<ParamField path="id_perfil" type="integer" required>
  The unique profile ID of the driver
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="message" type="string">
  Success or error message
</ResponseField>

<ResponseField name="data" type="object">
  Complete driver information
  
  <Expandable title="data properties">
    <ResponseField name="idPerfil" type="integer">
      Profile ID
    </ResponseField>
    
    <ResponseField name="idUsuario" type="integer">

    <ResponseField name="email" type="string">

      First name

      Second name

    <ResponseField name="apellido1" type="string">
      First surname
    </ResponseField>
    
    <ResponseField name="apellido2" type="string">
      Second surname
    </ResponseField>
    
    <ResponseField name="apellido3" type="string">
      Married name
    </ResponseField>
    
    <ResponseField name="telefono" type="string">
      Contact phone number
    </ResponseField>

      National ID number
    </ResponseField>

    </ResponseField>
    
    <ResponseField name="direccion" type="string">
      Residential address

    <ResponseField name="codigoEmpleado" type="string">
      Employee code
    </ResponseField>
    
    <ResponseField name="cargoNominal" type="string">
      Nominal position

    <ResponseField name="cargoFuncional" type="string">
      Functional position
    </ResponseField>
    
    <ResponseField name="observaciones" type="string">

    <ResponseField name="idDepartamentoAsignado" type="integer">

    <ResponseField name="disponibilidad" type="string">

      Last modification timestamp

  </Expandable>
</ResponseField>

## Status Codes

- `200`: Driver retrieved successfully
- `404`: Driver not found or has been deleted
- `500`: Internal server error

## Error Response Example

```json
{
  "success": false,
  "message": "Motorista no encontrado o ha sido eliminado",
  "data": null
}
```

## Use Cases

- Display driver profile page with complete information
- Retrieve driver details for editing forms
- Verify driver information before assigning missions
- Check license expiration dates and availability status
