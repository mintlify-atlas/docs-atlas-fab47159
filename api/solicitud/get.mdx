---
title: "Get Solicitud"
api: "GET /solicitud/{id}"
description: "Retrieve a specific solicitud by ID"
---

## Overview

Retrieves detailed information about a specific solicitud, including its current status, associated locations, and linked missions.

## Authentication

Requires valid authentication token.

## Path Parameters

<ParamField path="id_solicitud" type="integer" required>
  The unique ID of the solicitud to retrieve
</ParamField>

## Status Codes

<ResponseField name="200" type="OK">
  Solicitud retrieved successfully
</ResponseField>

  Solicitud with the specified ID does not exist or has been deleted
</ResponseField>

<ResponseField name="500" type="Internal Server Error">
  Server error
</ResponseField>
