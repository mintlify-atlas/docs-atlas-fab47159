## Create Mission

Creates a new mission associated with an approved request. The mission is created in `PROGRAMADA` (scheduled) state.

### Request Body

<ParamField body="idSolicitud" type="integer" required>
  ID of the associated request. The request must exist and be in `APROBADA` or `EN_PROCESO` state.
</ParamField>

<ParamField body="fechaProgramada" type="date" required>
  Scheduled date for the mission in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="horaSalidaProgramada" type="string" required>
  Scheduled departure time in `HH:MM:SS` or `HH:MM` format.
</ParamField>

<ParamField body="idLugarOrigen" type="integer">
  ID of the origin location. Must exist if provided.
</ParamField>

<ParamField body="idLugarDestino" type="integer">
  ID of the main destination location. Must exist if provided.
</ParamField>

<ParamField body="descripcion" type="string">
  Description of the mission. Maximum 500 characters.
</ParamField>

<ParamField body="horaLlegadaEstimada" type="string">
  Estimated arrival time in `HH:MM:SS` or `HH:MM` format.
</ParamField>

<ParamField body="observaciones" type="string">
  General observations about the mission.
</ParamField>

<ParamField body="paradas" type="array">
  List of stops for the mission.
  
  <Expandable title="Stop Object">
    <ParamField body="idLugarDestino" type="integer" required>
      ID of the destination location for this stop.
    </ParamField>
    
    <ParamField body="numeroParada" type="integer" required>
      Stop order number (must be >= 1).
    </ParamField>
    
    <ParamField body="descripcion" type="string">
      Description of the stop. Maximum 500 characters.
    </ParamField>
    
    <ParamField body="esParadaFinal" type="boolean" required>
      Whether this is the final stop of the mission.
    </ParamField>
    
    <ParamField body="fechaHoraLlegadaEstimada" type="string">
      Estimated arrival date and time in `YYYY-MM-DD HH:MM:SS` format.
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful.
</ResponseField>

<ResponseField name="message" type="string">
  Response message: "Misión creada exitosamente"
</ResponseField>

<ResponseField name="data" type="object">
  The created mission object with all details including stops.
</ResponseField>

### Validations

- **idSolicitud**: Required and must exist in `APROBADA` or `EN_PROCESO` state
- **fechaProgramada**: Required
- **horaSalidaProgramada**: Required
- Optional locations must exist if provided
- Initial state is always `PROGRAMADA`

<RequestExample>
```bash cURL
curl -X POST "https://api.example.com/mision" \
  -H "Content-Type: application/json" \
  -d '{
    "idSolicitud": 1,
    "idLugarOrigen": 1,
    "idLugarDestino": 2,
    "descripcion": "Traslado de personal al aeropuerto",
    "fechaProgramada": "2025-06-15",
    "horaSalidaProgramada": "08:00:00",
    "horaLlegadaEstimada": "09:30:00",
    "observaciones": "Llevar documentos",
    "paradas": []
  }'
```
