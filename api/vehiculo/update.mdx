---
title: 'Update Vehicle'
api: 'PUT /vehiculo/{id}'
description: 'Update an existing vehicle'
---

## Endpoint

<ParamField path="method" type="string" default="PUT">
  PUT
</ParamField>

<ParamField path="endpoint" type="string">
  /vehiculo/{id_vehiculo}
</ParamField>

## Description

Updates an existing vehicle record. All fields in the request body are optional - only send the fields you want to update. The vehicle must exist and be active.

## Path Parameters

<ParamField path="id_vehiculo" type="integer" required>
  Unique identifier of the vehicle to update
  
  **Example**: `1`
</ParamField>

## Request Body

<Info>
  All fields are optional. Only include the fields you want to update.
</Info>

<ParamField body="placa" type="string">
  Vehicle license plate. Must be unique if changed. Maximum 20 characters.
  
  **Example**: `"P503067"`
</ParamField>

<ParamField body="idClaseVehiculo" type="integer">
  ID of the vehicle class. Must exist and be active if changed.
  
  **Example**: `1`
</ParamField>

<ParamField body="idEstadoVehiculo" type="integer">
  ID of the vehicle state. Must exist and be active if changed.
  
  **Example**: `1`
</ParamField>

<ParamField body="idTipoCombustible" type="integer">
  ID of the fuel type. Must exist if provided.
  
  **Example**: `1`
</ParamField>

<ParamField body="idMotoristaPorDefecto" type="integer">
  ID of the default driver profile. Must exist if provided.
  
  **Example**: `10`
</ParamField>

<ParamField body="idDepartamentoAsignado" type="integer">
  ID of the assigned department. Must exist if provided.
  
  **Example**: `2`
</ParamField>

<ParamField body="marca" type="string">
  Vehicle brand. Maximum 100 characters.
  
  **Example**: `"TOYOTA"`
</ParamField>

<ParamField body="modelo" type="string">
  Vehicle model. Maximum 100 characters.
  
  **Example**: `"HILUX"`
</ParamField>

<ParamField body="anio" type="integer">
  Year of manufacture.
  
  **Example**: `2020`
</ParamField>

<ParamField body="color" type="string">
  Vehicle color. Maximum 50 characters.
  
  **Example**: `"BLANCO"`
</ParamField>

<ParamField body="numeroMotor" type="string">
  Engine number. Maximum 100 characters.
  
  **Example**: `"1GRFE123456"`
</ParamField>

<ParamField body="numeroChasis" type="string">
  Chassis number. Maximum 100 characters.
  
  **Example**: `"JT123456789012345"`
</ParamField>

<ParamField body="numeroVin" type="string">
  VIN number. Maximum 100 characters.
  
  **Example**: `"JT123456789012345"`
</ParamField>

<ParamField body="capacidadPasajeros" type="integer">
  Passenger capacity.
  
  **Example**: `5`
</ParamField>

<ParamField body="capacidadCombustibleGalones" type="number">
  Fuel capacity in gallons.
  
  **Example**: `80.0`
</ParamField>

<ParamField body="kilometrajeActual" type="integer">
  Current mileage.
  
  **Example**: `18000`
</ParamField>

<ParamField body="fechaUltimaRevision" type="string">
  Last revision date (YYYY-MM-DD format).
  
  **Example**: `"2024-03-15"`
</ParamField>

<ParamField body="fechaProximoMantenimiento" type="string">
  Next maintenance date (YYYY-MM-DD format).
  
  **Example**: `"2024-09-15"`
</ParamField>

<ParamField body="fechaTarjetaCirculacion" type="string">
  Circulation card date (YYYY-MM-DD format).
  
  **Example**: `"2023-12-01"`
</ParamField>

<ParamField body="numeroPolizaSeguro" type="string">
  Insurance policy number. Maximum 100 characters.
  
  **Example**: `"POL123456789"`
</ParamField>

<ParamField body="fechaVencimientoSeguro" type="string">
  Insurance expiration date (YYYY-MM-DD format).
  
  **Example**: `"2025-12-01"`
</ParamField>

<ParamField body="observaciones" type="string">
  Additional observations.
  
  **Example**: `"Mantenimiento realizado en marzo 2024"`
</ParamField>

## Validations

<Warning>
  **Important Validations**
  
  - Vehicle must exist and be active
  - If changing **placa**, it must be unique (not used by another active vehicle)
  - Any FK that is changed must exist and be active
  - **placa** is automatically converted to uppercase without spaces
</Warning>

## Partial Update Support

<Info>
  This endpoint supports partial updates. You can send only the fields you want to change, and all other fields will remain unchanged.
  
  For example, to only update the mileage:
  ```json
  {
    "kilometrajeActual": 18000
  }
  ```
</Info>
