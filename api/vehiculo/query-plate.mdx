## Endpoint

<ParamField path="method" type="string" default="GET">
  GET
</ParamField>

<ParamField path="endpoint" type="string">
  /vehiculo/consulta-placa/{placa}
</ParamField>

## Description

Queries vehicle information from the external license plate API without creating a record in the system. This endpoint is useful for previewing vehicle data before creation or for validation purposes.

<Info>
  Only returns results if the vehicle owner belongs to the institution (configurable via `PLACA_API_PROPIETARIO_FILTRO` environment variable). If the vehicle doesn't belong to the institution, a 403 error is returned.
</Info>

## Path Parameters

<ParamField path="placa" type="string" required>
  Vehicle license plate to query. Will be automatically converted to uppercase and spaces removed.
  
  **Validation**: Must have at least one initial letter and four digits minimum (format: `^[A-Z]{1,}[0-9]{4,}$`)
  
  **Example**: `"P503067"`
</ParamField>

## Validations

<Warning>
  **Plate Format Validation**
  
  The plate must match the pattern: at least one initial letter followed by at least four digits.
  
  - Valid: `P503067`, `ABC1234`, `XY56789`
  - Invalid: `1234`, `AB123`, `P50` (less than 4 digits)
</Warning>

## Configuration

<Info>
  **Environment Variable**: `PLACA_API_PROPIETARIO_FILTRO`
  
  This variable configures which institution owners are allowed. Only vehicles belonging to the configured institution will return data from this endpoint.
</Info>
