---
title: 'Create Vehicle'
api: 'POST /vehiculo'
description: 'Create a new vehicle with complete data'
---

## Endpoint

<ParamField path="method" type="string" default="POST">
  POST
</ParamField>

<ParamField path="endpoint" type="string">
  /vehiculo
</ParamField>

## Description

Creates a new vehicle record with complete data. This endpoint requires all vehicle information to be provided in the request body.

## Request Body

<ParamField body="placa" type="string" required>
  Vehicle license plate (unique). Automatically converted to uppercase and spaces removed.
  
  **Example**: `"P503067"`
</ParamField>

<ParamField body="idClaseVehiculo" type="integer" required>
  ID of the vehicle class. Must exist and be active in the catalog.
  
  **Example**: `1`
</ParamField>

<ParamField body="idEstadoVehiculo" type="integer" required>
  ID of the vehicle state. Must exist and be active in the catalog.
  
  **Example**: `1`
</ParamField>

<ParamField body="idTipoCombustible" type="integer">
  ID of the fuel type. Optional. Must exist if provided.
  
  **Example**: `1`
</ParamField>

<ParamField body="idMotoristaPorDefecto" type="integer">
  ID of the default driver profile. Optional. Must exist if provided.
  
  **Example**: `10`
</ParamField>

<ParamField body="idDepartamentoAsignado" type="integer">
  ID of the assigned department. Optional. Must exist if provided.
  
  **Example**: `2`
</ParamField>

<ParamField body="marca" type="string">
  Vehicle brand. Maximum 100 characters.
  
  **Example**: `"TOYOTA"`
</ParamField>

<ParamField body="modelo" type="string">
  Vehicle model. Maximum 100 characters.
  
  **Example**: `"HILUX"`
</ParamField>

<ParamField body="anio" type="integer">
  Year of manufacture.
  
  **Example**: `2020`
</ParamField>

<ParamField body="color" type="string">
  Vehicle color. Maximum 50 characters.
  
  **Example**: `"BLANCO"`
</ParamField>

<ParamField body="numeroMotor" type="string">
  Engine number. Maximum 100 characters.
  
  **Example**: `"1GRFE123456"`
</ParamField>

<ParamField body="numeroChasis" type="string">
  Chassis number. Maximum 100 characters.
  
  **Example**: `"JT123456789012345"`
</ParamField>

<ParamField body="numeroVin" type="string">
  VIN number. Maximum 100 characters.
  
  **Example**: `"JT123456789012345"`
</ParamField>

<ParamField body="capacidadPasajeros" type="integer">
  Passenger capacity.
  
  **Example**: `5`
</ParamField>

<ParamField body="capacidadCombustibleGalones" type="number">
  Fuel capacity in gallons.
  
  **Example**: `80.0`
</ParamField>

<ParamField body="kilometrajeActual" type="integer">
  Current mileage.
  
  **Example**: `15000`
</ParamField>

<ParamField body="fechaUltimaRevision" type="string">
  Last revision date (YYYY-MM-DD format).
  
  **Example**: `"2024-01-15"`
</ParamField>

<ParamField body="fechaProximoMantenimiento" type="string">
  Next maintenance date (YYYY-MM-DD format).
  
  **Example**: `"2024-07-15"`
</ParamField>

<ParamField body="fechaTarjetaCirculacion" type="string">
  Circulation card date (YYYY-MM-DD format).
  
  **Example**: `"2023-12-01"`
</ParamField>

<ParamField body="numeroPolizaSeguro" type="string">
  Insurance policy number. Maximum 100 characters.
  
  **Example**: `"POL123456789"`
</ParamField>

<ParamField body="fechaVencimientoSeguro" type="string">
  Insurance expiration date (YYYY-MM-DD format).
  
  **Example**: `"2025-12-01"`
</ParamField>

<ParamField body="observaciones" type="string">
  Additional observations.
  
  **Example**: `"Vehículo en excelente estado"`
</ParamField>

## Validations

<Warning>
  **Important Validations**
  
  - **placa**: Required, unique, converted to uppercase without spaces
  - **idClaseVehiculo**: Required, must exist and be active
  - **idEstadoVehiculo**: Required, must exist and be active
  - **Optional FKs**: If provided (idTipoCombustible, idMotoristaPorDefecto, idDepartamentoAsignado), they must exist
</Warning>

<Info>
  If a vehicle with the same plate was previously deleted, the system will reactivate that record instead of creating a new one.
</Info>
